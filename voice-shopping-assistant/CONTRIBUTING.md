# Contributing to Voice Shopping Assistant

We welcome contributions! Here's how to get started:

## Development Setup

1. Clone the repository
2. Run `./setup.sh` (Linux/Mac) or `setup.bat` (Windows)
3. Create a new branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Submit a pull request

## Code Style

- **Python:** PEP 8 (use `black` for formatting)
- **JavaScript:** Prettier format, ESLint rules
- **Components:** Functional components with hooks

## Testing

- Run backend tests: `cd backend && python -m pytest tests/`
- Frontend tests: `cd frontend && npm test`

## Pull Request Process

1. Update README.md with any new features
2. Include a clear description of changes
3. Ensure tests pass
4. One approval required before merge

## Bug Reports

Include:
- Steps to reproduce
- Expected vs actual behavior
- Browser/OS information
- Screenshots if applicable

Thank you for contributing!
